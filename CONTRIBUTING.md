# Contributing

Contributions are welcome!

## Setup

```bash
git clone https://github.com/amaar-mc/melodic-contour.git
cd melodic-contour
npm install
```

## Development

- `npm test` - run tests
- `npm run typecheck` - type check
- `npm run lint` - lint
- `npm run build` - build

## Guidelines

- All functions must be pure and strictly typed.
- All parameters must be explicit (no default values).
- Add tests for any new functionality.
- Follow the existing code style enforced by Biome.
- No runtime dependencies.
