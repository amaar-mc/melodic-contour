# Project Charter

## Purpose

melodic-contour provides a TypeScript implementation of melodic contour theory as formalized by Robert Morris, Michael Friedmann, and Elizabeth Marvin with Paul Laprade.

## Goals

- Correct, well-tested implementations of all core contour operations.
- Zero runtime dependencies.
- Dual ESM and CJS output with full TypeScript declarations.
- A natural companion to the pcset and tonnetz libraries.

## Non-Goals

- Real-time audio processing.
- MIDI parsing.
- Musical notation rendering.

## Audience

Computational musicologists, music theorists, and developers building music analysis tools.
